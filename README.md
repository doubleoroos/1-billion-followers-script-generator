## 🌟 Overview
**Rewrite Tomorrow Film Generator** is an AI-powered creative toolkit designed for filmmakers participating in the **1 Billion Summit AI Film Award**.  
The app turns high-level creative choices into a complete, production-ready blueprint for a 7–10 minute short film about positive future narratives.

Using the power of Google’s Gemini ecosystem (Gemini Pro, Imagen, Veo), the tool generates a full suite of narrative, visual, and cinematic assets from a single guided workflow.

---

## ✨ Key Features

### **Guided Creative Input**
Choose four creative pillars that shape the entire film:
- **Theme**
- **Narrative Tone**
- **Visual Style**
- **Emotional Arc**

---

### **Automatic Asset Generation**
In one workflow, the app generates:
- **Characters** — cast with personalities and roles  
- **Script** — a complete formatted screenplay  
- **Visual Outline** — scene-by-scene storyboard  
- **Moodboard** — Imagen-generated visual references  
- **Behind-the-Scenes Document** — AI workflow explanation matching competition criteria  

All elements are editable directly in the UI.

---

### **AI-Powered Visual Production**
Turn the storyboard into real visuals:

- **Preview Images** (static frames via image prompts)  
- **Video Clips** (fully animated scenes using Veo)  
- **Generate Full Film** — produce all clips automatically  

A prompt enhancer improves cinematic quality and visual consistency.

---

### **Efficient Workflow Tools**
- Bulk generation for prompts, images, or videos  
- Scene dependencies for continuity  
- Autosave for seamless editing  

---

## 🎥 How It Works (Pipeline)

1. **Envision** — Select theme, tone, style, emotion  
2. **Generate** — AI creates script, characters, and storyboard  
3. **Refine** — Edit text, prompts, and scenes  
4. **Visualize** — Generate preview images  
5. **Animate** — Render final video clips

## 📦 Installation

### 1. Clone the Repository
```bash
git clone https://github.com/doubleoroos/1-billion-followers-script-generator.git
cd 1-billion-followers-script-generator
````

### 2. Install Dependencies

Choose your environment:

**Node.js**

```bash
npm install
```

**Python**

```bash
pip install -r requirements.txt
```

**Flutter**

```bash
flutter pub get
```

---

### 3. Add Environment Variables

Create a `.env` file:

```
GEMINI_API_KEY=your_api_key_here
VEO_API_KEY=your_api_key_here
IMAGEN_API_KEY=your_api_key_here
```

> Never commit API keys to GitHub.

---

### 4. Run the App

**Web:**

```bash
npm run dev
```

**Flutter:**

```bash
flutter run
```

---

### 5. Build for Production

```bash
npm run build
```
