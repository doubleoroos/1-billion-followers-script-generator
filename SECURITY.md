This is a **private, closed-source project**.

## 🚫 Reporting Security Issues

### **Do NOT submit:**

* Pull requests
* Issues
* Fixes
* Vulnerability reports
* Feature requests
* Dependency updates

This project does **not** accept public or community submissions of any kind.

---

## 📩 If You Believe You Found a Critical Issue

Only the repository owner is authorized to evaluate security concerns.

If you must report a severe internal issue (e.g., exposure of private keys or unintended leaks), contact:

**[info@earthrising.space](mailto:info@earthrising.space)**
Subject: URGENT — Security Issue

Do *not* disclose the issue publicly or through GitHub Issues.

## 🔐 Access Control

Access to this repository is strictly limited.
You may **not**:

* Clone beyond authorized personal use
* Share code or files with third parties
* Export any part of the repository
* Mirror, fork, or publish contents elsewhere
* Upload the code to any AI or dataset training platform
* Integrate any file into another project without written permission

Any unauthorized access, reproduction, or distribution will result in enforcement measures.

---

## ⚠️ Private Keys & Credentials

Never commit or upload API keys, credentials, tokens, or secrets.
If keys are accidentally exposed, they must be rotated immediately.

The repository owner reserves the right to invalidate and replace compromised keys or API access at any time.
